2. в) Настройка HTTP протоколов и версий TLS/SSL

Для настройки протоколов и версий TLS/SSL в секции с описанием https узла
используются параметры "Protocols" и "SSLProtocols".
Возможные значения параметра "Protocols":
- Http1 - только HTTP/1.1
- Http2 - только HTTP/2
- Http3 - только HTTP/3
- Http1AndHttp2 - HTTP/1.1 и HTTP/2 (рекомендуемое)
- Http1AndHttp2AndHttp3 - все три версии HTTP
Для использования Http/2 требуется указывать TLS версии 1.2 и выше; для Http/3 -
TLS версии 1.3
Возможные значения параметра "SSLProtocols":
TLS 1.0 (устаревший, не рекомендуется)
TLS 1.1 (устаревший, не рекомендуется)
TLS 1.2 (безопасен, минимальная рекомендуемая версия)
TLS 1.3 (безопасен, рекомендуется)
Если параметр "SSLProtocols" не указан - по умолчанию используются настройки ОС.

6. Для настройки числа соединений необходимо в файле
...\WebSoftServer\xhttp_config.json добавить секцию Limits с параметрами:
"Limits": {
"MaxConcurrentConnections": 800, // Default is null (i.e. no maximum)
"MaxConcurrentUpgradedConnections": 800, // Default is null (i.e. no
maximum)
"MaxRequestBodySize": 536870912, // 512MB. Default is 30MB. Use
[RequestSizeLimit(100000000)] attribute to use more.
"MaxRequestHeaderCount": 100 // Default is 100
},
MaxConcurrentConnections"- задает максимальное число открытых соединений;
MaxConcurrentUpgradedConnections - задает максимальное число открытых,
обновленных соединений. Обновленное соединение — это подключение, которое
было переключено с HTTP на другой протокол, например WebSockets;
MaxRequestBodySize - задает максимально допустимый размер тела запроса в
байтах;
MaxRequestHeaderCount - задает максимально допустимое количество заголовков на
HTTP-запрос.
7. Управление заголовками а) Для того чтобы убрать определённые http-заголовки
из ответов сервера, необходимо в файле ...\WebSoftServer\xhttp_config.json добавить
секцию ResponseHeadersRules:
"ResponseHeadersRules": [ { "Host":"*", "Remove": ["Server"]

, где:
Host - указать, для каких адресов будет применяться правило(* - для всех);
Remove- перечислить заголовки, которые нужно убрать из ответов сервера.
