# Обзор базы знаний WebSoft HCM

База знаний «Установка и настройка WebSoft HCM» разбита на несколько крупных разделов. Структура и сами страницы загружаются динамически, поэтому для навигации необходимо использовать меню слева. Основные разделы перечислены ниже и кратко описаны по содержанию. Нумерация соответствует навигации в базе знаний【357488377425420†screenshot】.

## Основные разделы

| Раздел | Описание |
|---|---|
| **1. Установка** | Объясняет, как подготовить сервер для WebSoft HCM, содержит список системных требований, варианты установки (Docker, Linux, Windows), инструкции по развёртыванию дополнительных сервисов (WebSoftIDM, сервер записи VClass, Inventa, Telemetry), созданию тестового стенда и установке демонстрационного контента【693318755733946†screenshot】. |
| **2. Настройка** | Описывает конфигурацию после первичной установки: запуск как службы, настройка Kerberos/Windows Authentication, работа через HTTPS, подключение к внешним базам данных, мониторинг, кластеризация, виртуальный класс, настройки почтовых уведомлений, DCOM‑компоненты, публикация через Nginx, модуль прокторинга и др. | 
| **3. Обновление** | Содержит рекомендации по обновлению WebSoft HCM, инструкции для перехода с старых версий (2019.3, 2020.3, 2021.4 и т. д.), а также статьи о подготовке тестового стенда и показе заглушки на время работ【741554482264272†screenshot】. |
| **4. Обслуживание** | Включает инструкции по перезапуску WebSoft HCM, резервному копированию, восстановлению и другим операциям сопровождения. Также приводит советы при снижении производительности, администрировании внешних БД и восстановлении индексов Lucene【535997402157398†screenshot】. |
| **5. Типовые документы** | Раздел содержит шаблоны и образцы документов (например, письма и инструкции для пользователей). Эти элементы представлены в виде ссылок на DOCX‑файлы, поэтому текст нужно извлекать отдельно. |
| **6. Информационная безопасность** | Содержит рекомендации по защите WebSoft HCM: настройка HTTPS, Kerberos/NTLM, работа с сертификатами, шифрование данных и др. |
| **7. Архив** | Исторические документы и устаревшие инструкции. Следует использовать для справки, но ориентироваться на актуальные материалы из предыдущих разделов. |

## Навигация

* Чтобы открыть раздел, в меню слева нужно выбрать соответствующий пункт. Многие статьи содержат вложенные подразделы, доступные через подменю. При переходе название статьи отображается в заголовке и сверху справа есть кнопка «Экспорт в PDF», которая генерирует статический документ.
* Контент подгружается через JavaScript, поэтому для анализа страниц рекомендуется использовать браузер с поддержкой динамической загрузки или экспортировать PDF.

## Использование в этом пакете

В рамках этого install‑pack собраны только материалы, относящиеся к установке, системным требованиям и конфигурации сервера (Docker/Linux/Windows), внешним БД, HTTPS, запуску как службы, настройке почтовых уведомлений, резервному копированию, обновлению, безопасности и устранению типичных проблем. Остальные статьи (например, о разработке или работе с пользователями) не включены.
