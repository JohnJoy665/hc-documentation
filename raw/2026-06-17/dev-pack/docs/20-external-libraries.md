# Внешние библиотеки

Разработчики Websoft HCM могут использовать внешние библиотеки, поставляемые вместе с дистрибутивом.  В разделе «Внешние библиотеки» базы знаний описана библиотека `JS CMI5 Tool`, предназначенная для взаимодействия с системами, поддерживающими стандарт **cmi5**.

## JS CMI5 Tool

Библиотека `cmi5tool.js` входит в дистрибутив Websoft HCM и располагается в каталоге `scripts` портала.  Она позволяет отправлять и получать команды в формате cmi5 (современный стандарт для обмена данными о прохождении обучения).

### Подключение

Чтобы подключить библиотеку к пользовательской странице, добавьте в разметку следующий тег:

```html
<script src="/scripts/cmi5tool.js"></script>
```

### Использование

После подключения становятся доступными методы объекта `WT_CMI5`:

| Метод | Описание |
| --- | --- |
| `WT_CMI5.GetData(functionName, callback)` | Получает данные cmi5 для текущего пользователя.  Аргумент `functionName` определяет тип запрашиваемой информации. |
| `WT_CMI5.PutData(functionName, data)` | Отправляет данные (например, результаты теста) в формате cmi5.  Параметр `data` должен соответствовать спецификации. |

Пример вызова:

```javascript
// Получение данных о состоянии прохождения курса
WT_CMI5.GetData('GetState', function(state) {
    console.log('Текущее состояние:', state);
});

// Отправка результатов прохождения теста
WT_CMI5.PutData('SetResult', { score: 85, passed: true });
```

> Важно: библиотека предназначена для реализации стандартов cmi5 и предназначена для использования в SCORM‑/cmi5‑совместимых приложениях.  Перед отправкой данных убедитесь, что формат объекта соответствует требованиям спецификации cmi5【753438888989244†screenshot】【625473296571757†screenshot】.

На момент подготовки документации других внешних библиотек в разделе не найдено.  Возможные дополнительные библиотеки могут поставляться в будущих версиях и описаны в базе знаний.
