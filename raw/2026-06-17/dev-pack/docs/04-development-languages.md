# Языки разработки

WebSoft HCM поддерживает несколько языков для создания бизнес‑логики и расширений. Основными являются C# и серверный JavaScript. Для выборок данных используется язык XQuery, который выделен в отдельный раздел.

## C#

C# применяется для разработки удалённых действий, сервисных библиотек, серверных агентов и модулей, требующих строгой типизации или использования .NET‑библиотек. Код C# компилируется и выполняется на сервере. Разработка ведётся через OmniCode или встроенный редактор. Поддерживаются пространства имён `WebSoft.Common` и другие, поставляемые с платформой. Пример:

```csharp
using WebSoft.Common;

public class UserReportAction : IWebTutorAction
{
    public object Execute(object context)
    {
        // получение пользователя
        var user = tools.get_user(123);
        return "Hello, " + user.FullName;
    }
}
```

## Server‑Side JavaScript (SSJS)

Server‑Side JavaScript является основным языком для написания бизнес‑логики. Интерпретатор встроен в платформу SP‑XML. Код выполняется на сервере и имеет доступ к встроенным объектам (tools, user, process и др.)【529610742775508†screenshot】. Благодаря динамической типизации JavaScript удобен для быстрого создания скриптов. Пример функции:

```javascript
function calculateScore(userId) {
    var result = tools.get_user_score(userId);
    if (result > 80) return "Отлично";
    if (result > 50) return "Хорошо";
    return "Повторить";
}
```

Особенности SSJS:

- поддержка объектов и массивов, функций высшего порядка;
- доступ к встроенным функциям и библиотекам (см. раздел «Встроенные библиотеки»);
- возможность работы с XML и XQuery;
- отладка через OmniCode.

## XQuery

Для выборок из базы данных WebSoft HCM используется XQuery. Язык позволяет формировать сложные запросы к XML‑структурам, сортировать и фильтровать результаты. Подробности см. в разделе «XQuery».

> Важно:
> В исходной документации упоминаются и другие языки (например, XAML для интерфейса, SQL для внешних баз данных), но они используются в отдельных контекстах и рассматриваются в соответствующих разделах.
