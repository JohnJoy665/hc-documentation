# Библиотека Tools

Библиотека `Tools` — крупнейшая встроенная библиотека Websoft HCM.  Она реализована на платформе SP‑XML и содержит сотни функций, сгруппированных по категориям.  Многие серверные объекты (агенты, код библиотеки, удалённые действия) используют функции `Tools` для стандартных операций.

Ниже приведён обзор основных категорий и примеры функций.  Полный список доступен в справке системы; для каждой функции приведены синтаксис, параметры и возвращаемые значения.  В этом документе указаны только наиболее типичные функции с кратким описанием.

## Преобразование типов данных

| Функция | Назначение |
| --- | --- |
| `Int(value)` | Преобразует значение в целое число.  Если преобразование невозможно, генерируется исключение; для безопасного преобразования используйте `OptInt`【823943010247263†screenshot】. |
| `Real(value)` | Преобразует значение в действительное число (с плавающей точкой). |
| `DataType(value)` | Возвращает тип данных объекта (строка, число, логическое и т.п.). |
| `StrHexInt(intValue)` | Преобразует целое число в строку с шестнадцатеричным представлением. |
| `StrHexColor(color)` | Возвращает HEX‑код цвета по числовому представлению. |

## Работа со строками

| Функция | Назначение |
| --- | --- |
| `Len(str)` | Возвращает длину строки. |
| `Left(str, n)` | Возвращает первые `n` символов строки. |
| `Right(str, n)` | Возвращает последние `n` символов строки. |
| `Replace(str, old, new)` | Заменяет подстроку `old` на `new` во всей строке. |
| `Trim(str)` | Удаляет пробелы в начале и в конце строки. |

## Работа с датой и временем

| Функция | Назначение |
| --- | --- |
| `Now()` | Возвращает текущие дату и время в формате системы. |
| `DateAdd(date, interval, value)` | Прибавляет к дате указанное количество интервалов (дней, часов, минут). |
| `DateDiff(date1, date2, interval)` | Возвращает разницу между датами в выбранных единицах (дни, часы). |

## Работа с файлами и каталогами

| Функция | Назначение |
| --- | --- |
| `FileExists(path)` | Проверяет существование файла по указанному пути. |
| `FileCopy(src, dest)` | Копирует файл `src` в место `dest`. |
| `FileDelete(path)` | Удаляет файл. |
| `GetTempFileName(prefix)` | Генерирует временное имя файла с заданным префиксом. |

## XML‑ и XQuery‑утилиты

| Функция | Назначение |
| --- | --- |
| `XmlDocument(xmlString)` | Преобразует строку XML в объект DOM для последующей обработки. |
| `XmlNodeSelect(node, xpath)` | Выполняет выборку узлов по XQuery/XPath. |
| `XmlElement(name, attributes, content)` | Создаёт XML‑элемент с заданными атрибутами и содержимым. |

## Работа с курсами и учебными мероприятиями

| Функция | Назначение |
| --- | --- |
| `activate_course_to_event(course_id, event_id, employee_list)` | Назначает курс участникам указанного мероприятия【48942967479565†screenshot】. |
| `enroll_course(employee_id, course_code)` | Записывает сотрудника на курс по коду. |
| `get_course_status(employee_id, course_code)` | Возвращает статус прохождения курса сотрудником. |

## Прочие категории

* **Работа с JSON** – функции для сериализации и десериализации JSON‑структур.
* **Отправка сообщений** – методы для формирования и отправки писем через встроенный почтовый сервис.
* **Статистические функции** – расчёт средних значений, медианы и других метрик.
* **Управление курсами** – назначение электронного обучения, управление расписанием мероприятий, работа с результатами.

> Важно: некоторые функции библиотеки Tools помечены как устаревшие. Проверьте актуальность функции перед использованием, чтобы избежать её удаления в будущих версиях.
