# Редактор веб‑страниц Websoft HCM

В состав Websoft HCM входит встроенный редактор веб‑страниц, который позволяет создавать динамические страницы для портала, настраивать шаблоны и формы и подключать собственный JavaScript/CSS.  Редактор предназначен для разработчиков и администраторов и даёт возможность реализовать интерфейсы, не ограниченные стандартными экранными формами.

## Назначение

* **Создание кастомных страниц** – можно создавать страницы портала с произвольным содержимым, компонентами и логикой.
* **Использование шаблонов** – редактор поддерживает использование шаблонов и наследование от существующих страниц.
* **Встраивание кода** – разработчик может добавлять HTML, XAML, CSS и JavaScript для реализации нужного поведения.
* **Интеграция с API** – созданные страницы могут вызывать API Websoft HCM для получения данных и отправки команд.

## Основные возможности

* **Визуальный режим** – редактор предоставляет визуальный интерфейс для добавления и позиционирования элементов, настройки свойств и стилей.
* **Режим кода** – возможен переход в режим редактирования исходного кода (HTML/XAML/JS) для точной настройки разметки.
* **Предварительный просмотр** – можно посмотреть, как страница будет выглядеть для пользователя, и протестировать её функциональность.
* **Управление версиями** – редактор поддерживает сохранение нескольких версий страницы и откат при необходимости.
