# XQuery в WebSoft HCM

XQuery используется в WebSoft HCM для выборок данных из базы и формирует основу аналитических отчётов, экранных форм и удалённых действий. Запросы XQuery ориентированы на работу с XML‑структурами, что соответствует внутреннему представлению данных в системе.

## Синтаксис запроса【469683879944750†screenshot】

Базовая структура XQuery выглядит так:

```xquery
for $var1 in <выражение>,
    $var2 in <выражение>
where <условие>
order by <поле> ascending
return <возвращаемое значение>
```

- **Переменные** начинаются с символа `$` и объявляются после `for`. Несколько переменных разделяются запятыми.
- **Условия** в блоке `where` используют логические операторы `and` и `or`, а также сравнительные операторы `=`, `!=`, `<`, `>` и т. д.
- Внутри условий можно применять функции, например `contains($field, 'текст')` или `doc-contains($field, 'слово')` для полнотекстового поиска.
- Блок `order by` задаёт сортировку (по возрастанию или убыванию).  
- `return` определяет, какие элементы или значения вернутся в результате.

Пример запроса, выбирающего опубликованные курсы:

```xquery
for $c in courses
where $c/status = 'publish'
order by $c/name ascending
return $c
```

## Примеры запросов【337008561971132†screenshot】

1. **Получение сотрудников определённого подразделения:**

   ```xquery
   for $u in users
   where $u/branch_code = 'HR'
   return <user>{$u/id}{$u/full_name}</user>
   ```

2. **Фильтрация по нескольким условиям:**

   ```xquery
   for $u in users
   where $u/age > 25 and contains($u/position, 'менеджер')
   return $u
   ```

3. **Сортировка по нескольким полям:**

   ```xquery
   for $course in courses
   order by $course/created_date descending, $course/name ascending
   return $course
   ```

## Морфологические словоформы【368398257997868†screenshot】

Чтобы XQuery учитывал различные формы слов (падежи, склонения), можно включить поддержку словоформ. Для этого требуется установить свежий патч .NET Core и скопировать библиотеку словоформ в каталог `WebTutorServer`. После перезапуска сервера поисковые функции (`contains`, `doc-contains`) будут учитывать морфологические формы слов.  
Это повышает качество поиска по текстовым полям, особенно в русскоязычных системах.

## Поиск по точной фразе

В меню имеется подраздел «Поиск по конкретной фразе». В исходной документации этот раздел содержит советы по поиску точных фраз в XQuery, однако его содержание не было просмотрено во время текущего обхода.  
Обычно для поиска точной фразы используется функция `contains()` с передачей целого выражения, либо можно использовать XQuery 1.0 операторы `text() = 'фраза'` для точного совпадения.

## Рекомендации

- Старайтесь ограничивать выборки, добавляя условия `where` и `order by`, чтобы не загружать сервер лишними данными.
- Используйте функции для преобразования данных и поиска (`contains`, `starts-with`, `ends-with`).
- Для работы со словоформами убедитесь, что соответствующая библиотека установлена.

XQuery — мощный инструмент для выборки и трансформации данных в WebSoft HCM. Уделяйте внимание оптимизации запросов и используйте образцы в качестве шаблонов для собственных задач.
