# Библиотека Tools_web

`Tools_web` — специальная библиотека встроенных функций, которая предназначена для работы с веб‑интерфейсом портала Websoft HCM и обеспечения безопасности при отображении пользовательского контента.  В отличие от основной библиотеки `Tools`, функции `Tools_web` описываются в отдельном файле `wtv_tools_web.xml` и выполняют прикладные задачи, связанные с XSS‑защитой, обработкой параметров запросов и отправкой сообщений.

Согласно справке системы, библиотека включает следующие функции и группы функций【467471369149346†screenshot】:

| Функция | Описание |
| --- | --- |
| `check_sum_sid(sid, sum)` | Проверяет контрольную сумму параметра `sid` и возвращает `true`, если сумма совпадает. |
| `convert_xss(str)` | Конвертирует специальные символы в строке, защищая от XSS‑инъекций. |
| `xss_neutralize(str)` | Очищает строку от потенциально опасных HTML‑тегов и скриптов. |
| `send_message(title, body, recipients)` | Отправляет сообщение указанным получателям через внутреннюю систему уведомлений. |
| `parse_multiple_parameter(param)` | Разбирает строку с множеством значений параметра, например `id=1&1&2`, и возвращает список значений. |
| `create_captcha()` | Формирует CAPTCHA‑картинку и возвращает идентификатор, который нужно проверить на стороне сервера. |
| `validate_captcha(id, value)` | Проверяет введённый пользователем код CAPTCHA. |
| `nl2br(text)` | Заменяет символы перевода строки на теги `<br/>` для корректного отображения в HTML. |

> Важно: большинство функций `Tools_web` возвращают простые значения типа `string`, `boolean` или `list`, и должны вызываться на стороне сервера (например, из кода библиотеки или агента).  Перед выводом пользовательских данных в шаблоны рекомендуется применять `convert_xss` или `xss_neutralize` для предотвращения XSS‑атак.

Документация по каждой функции хранится в XML‑файле, который можно найти в каталоге `WebTutorServer/wtv` (обычно это `wtv_tools_web.xml`). При разработке следует сверяться с этим справочником для уточнения параметров и форматов возвращаемых данных.
