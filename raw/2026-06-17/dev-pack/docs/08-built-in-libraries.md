# Встроенные библиотеки Websoft HCM

Платформа Websoft HCM включает набор встроенных библиотек, которые предоставляют готовые функции и методы для работы с данными, файлами, строками, XML‑структурами, пользовательским интерфейсом и другими аспектами системы.  Использование этих библиотек позволяет ускорять разработку и избегать повторного написания однотипного кода.

В документе приведено краткое описание каждой встроенной библиотеки и перечислены основные категории функций, доступные для разработчиков.  Полный список функций находится в справочнике самой платформы; если требуются подробности по конкретной функции, обратитесь к официальной документации или к модулю справки в системе.

## Список встроенных библиотек

| Библиотека | Назначение | Пример источника |
| --- | --- | --- |
| **Tools** | Основная библиотека, содержащая большое количество функций для преобразования типов, работы со строками, датами, файлами, XML‑структурами, почтовыми уведомлениями, отправки сообщений, работы с курсами и обучающими мероприятиями. Функции сгруппированы по категориям и доступны из серверного кода и OmniCode. | Функция `activate_course_to_event` назначает курс участникам мероприятия【48942967479565†screenshot】; функция `Int` преобразует значения в целое число【823943010247263†screenshot】. |
| **Tools_library** | Небольшая библиотека для работы с материалами учебных библиотек. Содержит функции для конвертации прикреплённых файлов (например, преобразование PDF для просмотра на портале). | Функция `convert_pdf_libratry_material` конвертирует прикреплённый PDF в материал библиотеки【438568180020973†screenshot】. |
| **Tools_web** | Библиотека для работы с веб‑интерфейсом и безопасностью. Предоставляет функции для генерации и проверки контрольных сумм, нейтрализации XSS, отправки сообщений, обработки множественных параметров, работы с капчей и т.д. | Справочник `Tools_web` описывает функции `check_sum_sid`, `xss_neutralize`, `send_message` и др. 【467471369149346†screenshot】. |
| **Ms_tools** | Специальная библиотека для интеграции с продуктами Microsoft (например, Exchange/Outlook). > Важно: в каждой библиотеке функции могут отмечаться как устаревшие. Для новых разработок рекомендуется использовать актуальные методы, а старые функции постепенно выводить из эксплуатации.  Отдельный раздел «Deprecated functions» в библиотеке `Tools` содержит список устаревших функций, которые следует избегать.

### Категории функций библиотеки Tools

Ниже перечислены основные категории функций библиотеки `Tools`. Каждая категория содержит десятки методов. В разделе 9 приведён более подробный список функций с краткими описаниями.

| Категория | Описание |
| --- | --- |
| **Преобразование типов данных** | Функции для преобразования данных в различные типы: числа, строки, дробные значения, цвета в HEX, JSON и т.д. Пример: `Int(value)` – преобразует значение в целое число【823943010247263†screenshot】. |
| **Преобразование форматов** | Функции для конвертации данных между форматами: даты и времени, текстовых форматов, строк и числовых представлений. |
| **Работа со строками** | Методы для анализа, поиска, сравнения, разделения и модификации строк. |
| **Работа с датой и временем** | Функции для получения текущего времени, вычисления разницы между датами, добавления и вычитания интервалов. |
| **Работа с XML и XQuery** | Утилиты для создания, чтения и изменения XML‑структур, выполнения XQuery‑запросов. |
| **Файловая система** | Методы для создания, чтения и удаления файлов и папок, архивации и получения информации о файлах. |
| **Почтовые уведомления** | Функции для отправки почтовых сообщений и уведомлений через встроенный почтовый сервис. |
| **Работа с курсами и мероприятиями** | Методы для назначения курсов, создания событий, проверки статуса обучения. |
| **Другое** | Дополнительные функции для работы с ресурсами, экранными формами, web‑шаблонами, объектами системы, JSON‑структурами и т.д.

Встроенные библиотеки облегчают разработку на сервере, особенно при создании агентов, библиотек программного кода и удалённых действий.  Пользуйтесь функциями в соответствии с назначением и не забудьте проверять актуальность метода в справочнике.
