# Справочник функций библиотеки Tools

Библиотека `Tools` содержит сотни функций, сгруппированных по тематике.  Здесь приведён расширенный перечень категорий и примеры функций, основанный на структуре встроенного справочника【112763636933874†screenshot】.

## Категории и примеры

### Преобразование типов данных

| Функция | Описание |
| --- | --- |
| `Int(value)` | Преобразует значение в целое число【823943010247263†screenshot】. |
| `OptInt(value)` | Возвращает целое или `null`, если преобразование невозможно. |
| `Real(value)` | Преобразует значение в вещественное число. |
| `OptReal(value)` | Возвращает вещественное или `null`. |
| `DataType(value)` | Возвращает строковое представление типа данных. |
| `ObjectType(obj)` | Возвращает тип объекта системы. |

### Преобразование форматов

| Функция | Описание |
| --- | --- |
| `DateToStr(date, format)` | Преобразует дату в строку с заданным форматом. |
| `StrToDate(str, format)` | Преобразует строку в дату по формату. |
| `StrHexInt(intVal)` | Возвращает шестнадцатеричную строку для целого числа. |
| `StrSignedInt(intVal)` | Возвращает строку с знаком (±) для числа. |
| `StrSignedReal(realVal)` | Строка с знаком для вещественного числа. |

### Работа со строками

| Функция | Описание |
| --- | --- |
| `Len(str)` | Длина строки. |
| `Left(str, n)` | Первые `n` символов. |
| `Right(str, n)` | Последние `n` символов. |
| `Mid(str, pos, n)` | Подстрока длиной `n`, начиная с позиции `pos`. |
| `Replace(str, old, new)` | Замена подстроки. |
| `Trim(str)` | Удаляет пробелы по краям. |
| `Contains(str, substr)` | Проверяет наличие подстроки. |

### Даты и время

| Функция | Описание |
| --- | --- |
| `Now()` | Текущая дата и время. |
| `Today()` | Текущая дата без времени. |
| `DateAdd(date, interval, amount)` | Добавляет интервал (дни, часы, минуты). |
| `DateDiff(date1, date2, interval)` | Разница между датами. |
| `FormatDate(date, format)` | Форматирует дату по шаблону. |

### XML и XQuery

| Функция | Описание |
| --- | --- |
| `XmlDocument(xml)` | Возвращает DOM‑объект для строки XML. |
| `XmlNodeSelect(node, xpath)` | Выбирает узлы по XPath/XQuery. |
| `XmlElement(name, attrs, content)` | Создаёт элемент. |
| `XPathValue(node, path)` | Возвращает значение узла. |

### JSON

| Функция | Описание |
| --- | --- |
| `JsonParse(str)` | Преобразует строку JSON в объект. |
| `JsonSerialize(obj)` | Сериализует объект в строку JSON. |
| `JsonMerge(obj1, obj2)` | Объединяет два JSON‑объекта. |

### Файловая система

| Функция | Описание |
| --- | --- |
| `FileExists(path)` | Проверяет существование файла. |
| `FileCopy(src, dest)` | Копирует файл. |
| `FileDelete(path)` | Удаляет файл. |
| `GetTempFileName(prefix)` | Возвращает временное имя файла. |
| `ZipFolder(src, dest)` | Архивирует папку. |

### Почтовые уведомления

| Функция | Описание |
| --- | --- |
| `SendMail(to, subject, body)` | Отправляет письмо через SMTP. |
| `CreateNotification(empId, title, body)` | Создаёт уведомление в системе. |

### Работа с курсами и мероприятиями

| Функция | Описание |
| --- | --- |
| `activate_course_to_event(course, event, employees)` | Назначает курс участникам мероприятия【48942967479565†screenshot】. |
| `enroll_course(employee, course)` | Записывает сотрудника на курс. |
| `close_event(eventId)` | Завершает учебное мероприятие и фиксирует результаты. |

### Дополнительные категории

* **Строковые функции безопасности** – `HtmlEncode`, `HtmlDecode` и др.
* **Работа с XML‑каталогами** – получение, добавление и удаление записей.
* **Статистика и отчёты** – функции для сбора и обработки статистических данных.
* **Отладка и логирование** – методы `Log`, `DebugLog`, `WriteToFile`.
