# Функции `Tools_web`

Библиотека `Tools_web` содержит функции для работы с веб‑интерфейсом, защиты от XSS‑атак и обработки параметров URL.  Она хранится в файле `wtv_tools_web.xml` и предназначена для использования в серверном коде.

## Основные функции【467471369149346†screenshot】

| Функция | Описание |
| --- | --- |
| `check_sum_sid(sid, sum)` | Проверяет контрольную сумму параметра `sid`.  Используется для подтверждения того, что параметр не был подменён. |
| `convert_xss(str)` | Заменяет специальные символы в строке на HTML‑сущности для предотвращения XSS. |
| `xss_neutralize(str)` | Удаляет опасные теги и скрипты из строки. |
| `send_message(title, body, recipients)` | Отправляет уведомление указанным получателям.  Получатели задаются списком идентификаторов сотрудников. |
| `parse_multiple_parameter(param)` | Разбирает параметр URL с несколькими значениями (например, `id=1&2&3`) и возвращает массив. |
| `create_captcha()` | Создаёт изображение CAPTCHA и возвращает идентификатор и код. |
| `validate_captcha(id, value)` | Проверяет код, введённый пользователем, относительно созданной CAPTCHA. |
| `nl2br(text)` | Преобразует символы новой строки в `<br/>`, что полезно при отображении многострочных сообщений. |

> Большинство функций возвращают стандартные типы (`string`, `number`, `boolean`).  Перед выводом пользовательского ввода обязательно используйте функции `convert_xss` или `xss_neutralize` для предотвращения XSS‑уязвимостей.
