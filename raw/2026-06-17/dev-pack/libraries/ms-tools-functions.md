# Функции `Ms_tools`

Библиотека `Ms_tools` предназначена для интеграции Websoft HCM с продуктами Microsoft, такими как Exchange и Outlook.  В справке указано, что библиотека содержит четыре функции и описана в файле `ms_tools.xml`【222772313231018†screenshot】.  Полный перечень функций не был доступен; ниже приведены возможные функции на основе типовых сценариев:

| Функция | Возможное назначение |
| --- | --- |
| `send_exchange_invitation(eventId, employeeList)` | Отправляет приглашение на учебное мероприятие через Exchange календарь. |
| `update_exchange_event(eventId, changes)` | Обновляет сведения о мероприятии (время, место) в календаре Exchange. |
| `delete_exchange_event(eventId)` | Удаляет мероприятие из календаря Exchange. |
| `sync_outlook_contacts(employeeList)` | Синхронизирует контакты сотрудников с Outlook. |

> Поскольку оригинальный раздел не содержал детального описания библиотек Microsoft, этот список является предположительным.  Для точного использования библиотеки `Ms_tools` обратитесь к официальной документации или файлу `ms_tools.xml` в вашей установке.
