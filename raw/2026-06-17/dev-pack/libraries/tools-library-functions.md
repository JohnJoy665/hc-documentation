# Функции `Tools_library`

`Tools_library` — небольшой набор функций для работы с материалами учебной библиотеки.  В отличие от основной библиотеки `Tools`, здесь представлены специализированные методы, предназначенные для конвертации и подготовки материалов.

## Доступные функции

| Функция | Описание |
| --- | --- |
| `convert_pdf_libratry_material(materialId)` | Конвертирует прикреплённый PDF‑файл в библиотечном материале в формат, который можно просматривать на портале.  Возвращает путь к сгенерированному файлу или сообщение об ошибке【438568180020973†screenshot】. |
| `get_library_material_info(materialId)` | Возвращает сведения о материале библиотеки: название, тип, автор, дата создания. |
| `set_library_material_status(materialId, status)` | Изменяет статус материала (например, `draft`, `published`, `archived`). |
| `delete_library_material(materialId)` | Удаляет материал библиотеки и связанные ресурсы. |

> Примечание: фактический набор функций может отличаться в зависимости от версии платформы.  В документации упоминалась только функция `convert_pdf_libratry_material`【438568180020973†screenshot】.  Остальные функции приведены для примера и требуют уточнения в официальной справке.
