# Библиотеки программного кода – список

В разделе «Библиотеки программного кода» базы знаний перечислены многочисленные библиотеки, каждая из которых предоставляет набор функций для специфических задач.  Полный список виден в меню раздела【347106488841916†screenshot】.  В этом файле приведён перечень библиотек, который поможет ориентироваться при изучении исходного кода.

| Код библиотеки | Назначение | Примечание |
| --- | --- | --- |
| `libMain` | Основные функции и вспомогательные методы для ядра системы. | Часто используется другими библиотеками. |
| `libAnalytics` | Аналитика, расчёт KPI, построение отчётов. | Функции `calculateKPI`, `buildReport` и др. |
| `libApplication` | Создание и управление приложениями в Websoft HCM. | Помогает интегрировать приложения. |
| `libAppTests` | Тестовые функции и автотесты. | Используется для проверки приложений. |
| `libAssessment` | Оценка компетенций и результатов обучения. | Поддерживает тестирование. |
| `libAssessment360` | Дополнение для 360‑градусной оценки. | |
| `libAssessmentComp` | Оценка конкуренции и сравнительный анализ. | |
| `libAttendance` | Учёт присутствия на курсах и мероприятиях. | |
| `libAuth` | Функции для аутентификации и авторизации. | |
| `libAuxiliary` | Вспомогательные методы, используемые другими библиотеками. | |
| `libBusinessProcess` | Управление бизнес‑процессами и рабочими потоками. | |
| `libCourseAdmin` | Администрирование курсов и учебных программ. | |
| `libEmployee` | Работа с данными сотрудников. | |
| `libEvent` | Управление учебными мероприятиями. | |
| … | Дополнительные библиотеки, которые появляются в новых релизах. | |

> Для каждого модуля существует своя документация, доступная в OmniCode и в справке системы.  Чтобы вызвать функцию из библиотеки, используйте `tools.call_code_library_method('libName','methodName', params)`.
