# Отчёт о ходе работы по созданию dev‑pack Websoft HCM

Дата: 2026‑06‑11

## Что собрано полностью

- **Основные разделы**: подготовлены файлы `00-overview.md`, `01-agents.md`, `02-global-settings.md`, `03-omnicode.md`, `04-development-languages.md`, `05-server-side-javascript.md`, `06-csharp.md`, `07-xquery.md`.  Они содержат обобщённую информацию по каждому разделу, примеры кода и ссылки на источники.
- **Встроенные библиотеки**: создан документ `08-built-in-libraries.md`, который описывает библиотеки `Tools`, `Tools_library`, `Tools_web` и `Ms_tools`.  В отдельных файлах приведены справочники функций (`libraries/tools-functions.md`, `tools-library-functions.md`, `tools-web-functions.md`, `ms-tools-functions.md`).
- **Кодовые библиотеки**: `11-code-libraries.md` содержит описание концепции библиотек, примеры вызова функций и список популярных библиотек.  В `libraries/code-libraries.md` представлен расширенный перечень библиотек.
- **Интерфейс и XAML**: файлы `12-system-interface.md`, `13-xaml-reference.md` и `xaml/xaml-elements.md` описывают принципы создания интерфейсов, структуру XAML‑разметки и перечень элементов.
- **API**: подготовлены файлы `14-api-overview.md`, `15-api-v1.md`, `16-api-v3.md`.  Они содержат общую информацию по версиям API, схему организации доступа, примеры запросов и рекомендации.
- **Дополнительные разделы**: созданы документы `17-web-page-editor.md`, `18-design-guidelines.md`, `19-caching.md`, `20-external-libraries.md`, `21-common-mistakes.md`, `22-examples.md`.
- **Файлы поддержки**: `README.md`, `llms.txt`, `llms-full.md`, `sources.json` и `progress-report.md` подготовлены.  В каталоге `api` содержатся справочные файлы для API, а в `raw/collected-pages.json` хранится перечень посещённых страниц (формируется автоматически).

## Что собрано частично

- **Server‑Side JavaScript**: собран общий обзор, формы `.xmd` и `.xms`, категории встроенных функций.  Однако детальная документация по каждой функции и объекту языка не включена из‑за объёма.
- **Библиотека Tools**: приведены основные категории и примеры функций, но не весь каталог.  Описание каждой функции требует дополнительной обработки.
- **Библиотеки программного кода**: приведён общий список библиотек и примеры вызовов, но для каждой библиотеки не составлены отдельные файлы.  Требуется дальнейшее изучение содержимого libMain, libAnalytics и других библиотек.
- **API v3**: собран обзор, структура доступа и примеры вызовов, однако не все подразделы (например, «Вызов методов API», «Пример программирования») детально проработаны.  Дополнительные примеры можно добавить из официальной спецификации.
- **Интерфейс и XAML**: список элементов приведён частично; полная таблица атрибутов для каждого элемента требует более глубокого изучения.
