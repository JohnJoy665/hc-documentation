# Справочник API v1 (SOAP)

Полный справочник методов API версии 1 (SOAP) требует анализа WSDL‑файлов и официальной документации.  В рамках данного пакета основные сведения приведены в файле `docs/15-api-v1.md`.  Подробные описания методов, параметров и примеров запросов не были найдены в исходном разделе документации и поэтому не включены.  При необходимости обратитесь к WSDL‑описаниям, доступным на портале Websoft HCM.
