# Websoft HCM Development Pack

Этот пакет содержит AI‑дружественную документацию для программистов, работающих с платформой **Websoft HCM / WebTutor**.  Он построен на основе раздела «Websoft HCM: среда разработки» из базы знаний WebSoft HCM TechNet и предназначен для упрощения работы с ChatGPT, Cursor/VS Code и системами RAG (MCP) при разработке решений на платформе.

## Что входит в пак‑файл

- **Документы в каталоге `docs/`** – основные разделы и справочники по языкам разработки, OmniCode, XQuery, встроенным библиотекам, системному интерфейсу, XAML‑разметке, API и другим темам. Каждый файл содержит фронт‑маттер с метаданными, краткое описание, синтаксис, правила использования и примеры кода.
- **Подкаталоги `api/`, `libraries/` и `xaml/`** – более детальные справочники по API (версии 1 и 3), встроенным библиотекам (`Tools`, `Tools_web`, `Ms_tools`) и XAML‑элементам.
- **Файл `llms.txt`** – краткая карта документации, которую удобно использовать в ChatGPT и RAG‑системах для навигации.
- **Файл `llms-full.md`** – единый консолидированный документ, содержащий основные разделы пакета. Его удобно загружать в ChatGPT для контекстной поддержки при разработке.
- **`sources.json`** – список всех исходных страниц, обработанных при сборке пакета, со статусом обработки.
- **`progress-report.md`** – отчёт о ходе работы с указанием обработанных и оставшихся разделов.
- **`raw/collected-pages.json`** – машинно‑читаемый перечень URL, использованных для подготовки содержимого, и кратких заметок.

## Как использовать

### ChatGPT

1. Загрузите файл `llms-full.md` в свой сеанс ChatGPT (или используйте указатель на него, если RAG‑механизм подключен к базе файлов).
2. Для навигации по пакету используйте карту из `llms.txt` или напрямую открывайте нужные markdown‑файлы из каталога `docs/`.
3. В вопросах к ChatGPT ссылаться на названия файлов и разделов – это поможет подобрать правильную информацию.

### Cursor / VS Code

1. Распакуйте архив в удобный каталог и добавьте его в рабочее пространство VS Code.
2. Просматривайте файлы в каталоге `docs/` и соответствующие примеры кода, копируйте конфигурации и фрагменты в свои проекты.
3. Используйте встроенный поиск VS Code для быстрого перехода к нужной функции или параметру.

### MCP / RAG‑платформы

1. Сконфигурируйте вашу RAG‑систему для индексирования файлов из `websoft-hcm-dev-pack/`.
2. Добавьте метаданные, например `source_urls` и `title`, чтобы система могла возвращать ссылки на первоисточники.
3. Пользователи RAG‑системы смогут задавать вопросы о Websoft HCM и получать ответы на основе документации из пакета.

## Ограничения

* Пакет собран вручную агентом и может содержать только те статьи и примеры, которые были доступны без авторизации. Если отдельные страницы были недоступны, это отражено в `sources.json`.
* Встроенные библиотеки содержат десятки функций. Документация описывает основные категории и примеры, но не включает полный справочник по каждому методу.
* API‑справочники в каталоге `api/` приведены в обзорной форме; за подробностями обращайтесь к официальной OpenAPI‑спецификации.
* Некоторые разделы, отмеченные как pending в отчёте, требуют дополнительного исследования.

Надеемся, что этот пакет поможет вам быстрее разобраться в разработке под Websoft HCM и позволит эффективно использовать ChatGPT и инструменты наподобие Cursor для работы с кодом.
