# Список элементов XAML Websoft HCM

Данный файл содержит таблицу элементов XAML, доступных для разработки интерфейсов в Websoft HCM.  Для каждого элемента указаны краткое назначение и основные параметры.  Этот список основан на структуре раздела «Описание XAML разметки» базы знаний и может быть неполным.

| Элемент | Описание | Основные атрибуты |
| --- | --- | --- |
| `Action` | Определяет удалённое действие, выполняемое при наступлении события; обычно размещается в начале формы【575597786908733†screenshot】. | `Name`, `Source`, `Success`, `Failure`, `Parameters` |
| `Button` | Кнопка для запуска действия или команды. | `Caption`, `Action`, `Image`, `Style`, `Width`, `Height` |
| `Label` | Выводит текстовые строки (статические или динамические). | `Caption`, `Style`, `Width` |
| `Grid` | Контейнер, обеспечивающий сеточную компоновку элементов. | `Columns`, `Rows`, `Width`, `Height` |
| `StackPanel` | Контейнер, размещающий дочерние элементы по горизонтали или вертикали. | `Orientation`, `Spacing` |
| `Panel` | Универсальный контейнер для свободного размещения элементов; позволяет управлять позиционированием. | `Width`, `Height`, `Style` |
| `ListBox` | Отображает список значений с возможностью выбора. | `Items`, `SelectionMode`, `OnChange` |
| `ComboBox` | Выпадающий список для выбора одного значения. | `Items`, `SelectedItem`, `OnChange` |
| `CheckBox` | Элемент для выбора булевого значения. | `Caption`, `Checked`, `OnChange` |
| `RadioButton` | Группа взаимно исключающих переключателей. | `GroupName`, `Caption`, `Checked` |
| `TextBox` | Поле ввода текста. | `Value`, `MaxLength`, `Placeholder`, `OnChange` |
| `Image` | Отображает изображение из ресурса или по URL. | `Source`, `Width`, `Height`, `Stretch` |
| `Menu` | Выпадающее меню с пунктами `MenuItem`. | `Items`, `OnClick` |
| `BackgroundImage` | Задаёт фоновое изображение для формы или элемента【449033167658910†screenshot】. | `Source`, `Mode` |

> Примечание: список элементов приведён для ориентира.  Для каждого элемента существует множество дополнительных параметров и событий, описанных в официальной документации Websoft HCM.
