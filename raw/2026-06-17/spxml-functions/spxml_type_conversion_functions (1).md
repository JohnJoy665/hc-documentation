# Преобразование типов данных

## `Compare()`

Функция `Compare()` сравнивает два произвольных значения.

Возвращает:

- `0`, если значения равны;
- положительное число, если первое значение больше второго;
- отрицательное число, если первое значение меньше второго.

### Синтаксис

```txt
Compare( value1, value2 )
```

### Аргументы

| Аргумент | Тип | Описание |
|---|---|---|
| `value1` |  | Значение 1 |
| `value2` |  | Значение 2 |

---

## `DataType()`

Функция `DataType()` возвращает строку, содержащую тип данных переданного значения.

Возможные значения:

- `"undefined"`
- `"null"`
- `"string"`
- `"integer"`
- `"real"`
- `"bool"`
- `"date"`
- `"object"`

### Синтаксис

```txt
DataType( value )
```

### Аргументы

| Аргумент | Тип | Описание |
|---|---|---|
| `value` |  | Значение |

### Смотри также

- `ObjectType()`

---

## `IsEmptyValue()`

Функция `IsEmptyValue()` проверяет, является ли значение `undefined`, `null` либо пустой строкой.

Позволяет писать универсальный код, не зная точный тип данных переданного аргумента.

---

## `ObjectType()`

Функция `ObjectType()` возвращает тип переданного объекта в низкоуровневом виде.

Если переданное значение не является объектом, функция возвращает пустую строку.

Примеры возможных значений:

- `"JsObject"`
- `"JsArray"`
- `"XmElem"`
- `"XmMultiElemObject"`

и др.

### Синтаксис

```txt
ObjectType( value )
```

### Аргументы

| Аргумент | Тип | Описание |
|---|---|---|
| `value` |  | Значение |

---

## `OptReal()`

Функция `OptReal()` преобразует значение аргумента к вещественному числу.

Если аргумент не может быть преобразован к вещественному числу, функция возвращает `undefined`, либо `defaultVal`, если оно указано.

### Синтаксис

```txt
OptReal( arg )
OptReal( arg, defaultVal )
```

### Аргументы

| Аргумент | Тип | Описание |
|---|---|---|
| `arg` |  | Целое число, вещественное число или строка, содержащая вещественное число |
| `defaultVal` | optional | Значение по умолчанию |

### Пример

```txt
OptReal( 12.6 )
OptReal( '12.6', null )
```

---

## `Real()`

Функция `Real()` преобразует значение аргумента к вещественному числу.

Если аргумент не может быть преобразован к вещественному числу, функция отрабатывает с ошибкой.

### Синтаксис

```txt
Real( arg )
```

### Аргументы

| Аргумент | Тип | Описание |
|---|---|---|
| `arg` |  | Целое число, вещественное число или строка, содержащая вещественное число |

### Пример

```txt
Real( 12.6 )
Real( '12.6' )
```

## `Int()`

Функция `Int()` преобразует значение аргумента к целому числу.

Если преобразование выполнить невозможно, функция возвращает ошибку.

### Синтаксис

```txt
Int( arg )
```

### Аргументы

| Аргумент | Тип | Описание |
|---|---|---|
| `arg` |  | Целое число, вещественное число или строка, содержащая число |

### Пример

```txt
Int( 123 )
Int( '123' )
Int( 123.45 )
```

---

| `arg` | `Any` | Аргумент |

### Результат

`Any`

## `StrHexColor()`

Функция `StrHexColor()` преобразует десятичное обозначение цвета `RGB` в шестнадцатиричное, принятое в формате HTML.

### Аргументы

| Аргумент | Тип | Описание |
|---|---|---|
| `color` | `String` | Десятичное обозначение цвета |

### Результат

`String`

### Пример

```txt
StrHexColor( '128,128,128' )
```

вернет:

```txt
'808080'
```

## `StrHexInt()`

Функция `StrHexInt()` возвращает строку, содержащую аргумент в шестнадцатиричном виде на 64 бита.

### Аргументы

| Аргумент | Тип | Описание |
|---|---|---|
| `num` | `Integer` | Число, которое нужно преобразовать |

### Результат

`String`

### Пример

```txt
StrHexInt( 1000 )
```

вернет:

```txt
'00000000000003E8'
```

## `StrInt()`

Функция `StrInt()` преобразует неотрицательный целочисленный аргумент в строку.

### Синтаксис

```txt
StrInt( arg )
StrInt( arg, digitsNum )
StrInt( arg, digitsNum, addGroupDelim )
```

### Аргументы

| Аргумент | Тип | Описание |
|---|---|---|
| `arg` | `integer` | Целочисленный аргумент |
| `digitsNum` | `integer optional` | Минимальное число символов в строке. Недостающие символы компенсируются нулями перед числом |
| `addGroupDelim` | `bool optional` | Разделять тысячные разряды пробелами |

### Пример

```txt
StrInt( 11500 )
```

вернет:

```txt
'11500'
```

```txt
StrInt( 11500, 6 )
```

вернет:

```txt
'011500'
```

```txt
StrInt( 11500, 0, true )
```

вернет:

```txt
'11 500'
```

## `StrIntZero()`

Функция `StrIntZero()` преобразует целочисленный аргумент в строку аналогично функции `StrInt()`.

Если значение целочисленного аргумента равно `0`, функция возвращает `"-"`.

### Аргументы

| Аргумент | Тип | Описание |
|---|---|---|
| `arg` | `Integer` | Целочисленный аргумент |
| `digitsNum` | `Integer optional` | Минимальное число символов в строке |
| `addGroupDelim` | `Bool optional` | Разделять тысячные разряды пробелами |

### Результат

`String`

## `StrReal()`

Функция `StrReal()` преобразует вещественный аргумент в строку.

Если аргумент равен `null` либо пустой строке, функция возвращает пустую строку.

### Синтаксис

```txt
StrReal( arg )
StrReal( arg, maxPrecision )
StrReal( arg, maxPrecision, addGroupDelim )
StrReal( arg, maxPrecision, addGroupDelim, options )
```

### Аргументы

| Аргумент | Тип | Описание |
|---|---|---|
| `arg` | `real` | Вещественный аргумент |
| `maxPrecision` | `integer optional` | Максимальное число знаков после запятой. По умолчанию `6` |
| `addGroupDelim` | `bool optional` | Разделять тысячные разряды пробелами |
| `options` | `object optional` | Базовый объект, содержащий опции вызова |

### Описание

Поддерживаются следующие опции:

| Опция | Тип | Описание |
|---|---|---|
| `ReplaceZero` | `bool` | Если вещественный аргумент равен нулю, возвращать символ `"-"` |

### Пример

```txt
StrReal( 90154.249 )
```

вернет:

```txt
'90154.249'
```

```txt
StrReal( 90154.249, 2 )
```

вернет:

```txt
'90154.25'
```

```txt
StrReal( 90154.249, 2, true )
```

вернет:

```txt
'90 154.25'
```

## `TextInt()`

Функция `TextInt()` преобразует целое число в словесное строковое представление.

Работает только для русского языка.

### Аргументы

| Аргумент | Тип | Описание |
|---|---|---|
| `arg` | `Integer` | Числовой аргумент |
| `gender` |  | Род: `0` — мужской, `1` — женский. Необязательный аргумент |

### Результат

`String`

### Пример

```txt
TextInt( 121 )
```

возвращает:

```txt
'сто двадцать один'
```

```txt
TextInt( 121, 1 )
```

возвращает:

```txt
'сто двадцать одна'
```
