# SP-XML: Работа с элементами XML

## Список функций

- `CreateElemByFormElem()`
- `CreateElem()`
- `CreateDynamicElem()`
- `GetForeignElem()`
- `GetOptForeignElem()`
- `GetFailedForeignElem()`
- `LoadElemsFromStr()`
- `ExportElemsToStr()`

---

## `CreateElemByFormElem()`

Создает XML-элемент по заданному элементу формы. Созданный элемент не имеет родительского элемента. Редко используемая функция.

### Синтаксис

```spxml
CreateElemByFormElem( formElem )
```

### Аргументы

| Аргумент | Тип | Описание |
|---|---|---|
| `formElem` | `object` | Объект вида `XmlFormElem` |

### Пример

```spxml
attachment = CreateElemByFormElem( candidate.attachments.FormElem[0] );
candidate.attachments.AddChildElem( attachment );
```

Эквивалентно:

```spxml
attachment = candidate.attachments.AddChild();
```

## `CreateElem()`

Создает XML-элемент заданному по фрагменту формы. Созданный элемент не имеет родительского элемента.

Функция используется для управления сложными структурами в оперативной памяти компьютера (как альтернатива javascript object), а так же для формирования XML для внешнего использования.

### Аргументы

| Аргумент | Тип | Описание |
|---|---|---|
| `formUrl` | `String` | url xmd-файла, в котором находится форма |
| `elemPath` | `String` | пусть к фрагменту формы |

### Результат

Объект `XmElem`.

### Пример

```spxml
CreateElem( '//base2/base2_user.xmd', 'user' );
```

## `CreateDynamicElem()`

Создает динамический (без привязки к форме) XML-элемент. Созданный элемент не имеет родительского элемента.

### Аргументы

| Аргумент | Тип | Описание |
|---|---|---|
| `name` | `String` | имя элемента |
| `type` | `String` | тип данных XML-элемента |

### Результат

Объект `XmElem`.

### Пример

```spxml
CreateDynamicElem( 'elem_count', 'integer' );
```

## `GetForeignElem()`

Выдает целевой элемент массива по значению первичного ключа. Смотри так же `GetOptForeignElem`.

Действие функции несколько отличается от функции `ArrayOptFindByKey()` за счет поддержки рекурсивных массивов XML-элементов.

Кроме этого, в новой объектной модели функция никогда не завершается с ошибкой, если соотвевтвующей элемент не найден, а возвращает пустой псевдо-элемент (режим терпимости к незаполенным и битым ссылкам).

### Аргументы

| Аргумент | Тип | Описание |
|---|---|---|
| `array` |  | массив объектов |
| `keyVal` | `any` | значение ключевого элемента |

### Результат

Объект (как правило `XmlElem`).

### Пример

```spxml
GetForeignElem( event_types, 'interview' )
```

## `GetOptForeignElem()`

Выдает целевой элемент массива по значению первичного ключа. Если соответсвующей элемент не найден, возвращается `undefined`.

Смотри так же `GetForeignElem()`.

Действие функции несколько отличается от функции `ArrayOptFindByKey()` за счет поддержки рекурсивных массивов XML-элементов.

### Аргументы

| Аргумент | Тип | Описание |
|---|---|---|
| `array` |  | массив объектов |
| `keyVal` | `any` | значение ключевого элемента |

### Результат

Объект (как правило `XmlElem`).

### Пример

```spxml
GetOptForeignElem( event_types, 'interview' )
```

---

## `GetFailedForeignElem()`

Создает новый пустой элемент массива, не добавляя его в в сам массив. Используется для отработки "битых ссылок" на элементы массива и ссылок на удаленные элементы массива. Смотри так же `GetOptForeignElem` и `GetForeignElem`.

В текущей реализации массив может быть только каталогом.

### Аргументы

| Аргумент | Тип | Описание |
|---|---|---|
| `array` |  | массив элементов |

### Результат

Объект `XmlElem`.

---

## `LoadElemsFromStr()`

Загружает строку в массив XML-элементов. Используется при обработке параметров, полученных от внешних процедур и плагинов. Смотри также `ExportElemsToStr`.

### Аргументы

| Аргумент | Тип | Описание |
|---|---|---|
| `arg1` | `string` | строка |

### Результат

`Array`.

---

## `ExportElemsToStr()`

Загружает массив XML-элементов в строку. Используется для передачи параметров в плагины и другие внешние процедуры. Смотри так же `LoadElemsFromStr`.

### Аргументы

| Аргумент | Тип | Описание |
|---|---|---|
| `arg1` | `array of objects` | массив XML-элементов |

### Результат

`string`.
