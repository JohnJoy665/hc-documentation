# Пакет «Websoft HCM: среда разработки» – путеводитель по разделу

Этот архив содержит путеводитель по разделу «Websoft HCM: среда разработки» из базы знаний Websoft HCM TechNet.  
Путеводитель создан на основе навигации по сайту и кратких описаний основных тем и статей, найденных в этом разделе.  
Внутренние страницы не копируются полностью: здесь представлена только структура меню и краткие аннотации для ориентации.

## Состав архива

- `00-guide-map.md` — основная карта раздела. Она отражает иерархию меню (основные темы и вложенные статьи), приведены ссылки на источник и краткие описания каждой темы.
- `sources.json` — машинно‑читаемый список всех найденных пунктов: названия, ссылки, родительские разделы, краткое описание и статус загрузки.
- `next-steps.md` — рекомендации по дальнейшему сбору полной документации. На основе карты предлагается порядок, в котором лучше обрабатывать темы.

## Ограничения

Путеводитель создан в ходе частичного просмотра раздела. Некоторые темы и вложенные статьи могли остаться неисследованными или не открыться из‑за технических ограничений. Внутри файлов отмечены только те материалы, которые удалось увидеть.  
Полноценная AI‑friendly документация будет собрана на следующем этапе: нужно пройти по каждому пункту карты, открыть полные статьи, извлечь содержимое и подготовить удобный для обучения пак.  
Настоящий архив предназначен для первоначальной ориентации в содержимом и планирования дальнейшей работы.
