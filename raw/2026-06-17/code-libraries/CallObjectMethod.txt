Функция CallObjectMethod() позволяет вызвать метод абстрактного объекта по имени
динамическим способом. Таким образом, вызов CallObjectMethod( object, 'Test', [arg1,
arg2] ) эквивалентен выражению object.Test( arg1, arg2 ).
Синтаксис
CallObjectMethod( object, methodName, argsArray )
