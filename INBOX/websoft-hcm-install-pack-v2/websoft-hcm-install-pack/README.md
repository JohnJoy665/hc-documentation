# Руководство к AI‑friendly install‑pack

Этот репозиторий содержит структурированный набор инструкций по развёртыванию WebSoft HCM (WebTutor) для администраторов. Файлы подготовлены в формате, удобном для дальнейшей обработки языковыми моделями (LLM) и инструментами вроде Cursor.

## Состав пакета

* **docs/** — ключевые материалы. Каждый файл начинается с frontmatter с метаданными (title, source_urls, collected_at и status). Папка содержит 14 файлов: от обзора (00) до инструкций по устранению неполадок (13).
* **raw/pages.json** — список всех источников с указанием, были ли успешно извлечены тексты и есть ли вложения.
* **llms.txt** — краткая карта, какую информацию искать в каком файле.
* **llms-full.md** — консолидированная версия всех markdown‑файлов (создаётся на финальном этапе).

## Как использовать этот пакет

1. **Изучение документов.** Начните с файла `docs/00-overview.md`, чтобы ознакомиться со структурой базы знаний и понять, какие разделы покрывают интересующие вас задачи.
2. **Поиск нужного раздела.** В `llms.txt` указано, какой документ содержит пошаговые инструкции по конкретной теме: установка через Docker, развёртывание на Linux/Windows, настройка HTTPS, работа с внешними базами данных и т. п.
3. **Работа в ChatGPT или Cursor.** Открыв соответствующий файл в редакторе (например, Cursor), можно задавать LLM вопросы по содержимому. Поскольку файлы имеют чёткую структуру и ссылки на исходные статьи, модель сможет быстро найти нужную информацию.
4. **Использование MCP (Markdown Content Processor).** Если вы используете MCP для генерации документации или внутреннего портала, просто загрузите файлы из `docs/` — frontmatter позволит автоматически построить оглавление и добавить метаданные.

## Ограничения

* Пакет содержит только те разделы базы знаний, которые напрямую касаются установки, настройки сервера, системных требований, Docker/Linux/Windows, внешней БД, HTTPS, запуска как службы, почтовых уведомлений, резервного копирования, обновления, безопасности и типичных проблем. Остальные материалы (разработка, пользовательские инструкции и т. п.) не собирались.
* Если в статьях встречались ссылки на вложенные файлы (DOCX, PDF и т. п.), они зафиксированы в `raw/pages.json`, но сами вложения не извлекались. Для работы с ними потребуется вручную скачать файлы и конвертировать их в текст.
* Материалы основаны на версии базы знаний WebSoft HCM, актуальной на момент сбора (10 июня 2026 года). Если документация будет обновлена, рекомендуется повторить сбор.
