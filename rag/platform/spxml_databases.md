# SP-XML базы данных и URL документов

Основные данные хранятся как XML-документы внутри баз данных.

Типовые базы:

- `data` - основная серверная база;
- `data_local` - пользовательская локальная база;
- `trash` - база корзины.

Статические документы доступны через `x-local://`, например:

```text
x-local://data/static/global_settings.xml
```

Объектные документы в новой модели доступны через `x-db-obj://`. WebTutor, по источнику, использует старую объектную модель, где объектные документы адресуются через `x-local://`.
