# Каталоги-связки

Каталог-связка - это представление (`view`), сформированное стандартными средствами через объединение двух или более стандартных каталогов по набору полей.

Каталоги-связки используются для облегчения и ускорения работы со связанными каталогами.

В `XQuery` каталог-связка используется так же, как обычный каталог:

```js
XQuery( 'for $elem in group_collaborators return $elem' );
```

Набор каталогов-связок зависит от версии WebTutor. Источник фиксирует состав для версии `331 (2021.2.3.331)`.
