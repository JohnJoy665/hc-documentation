# Объектная модель, XMD и каталоги

WebTutor / WebSoft HCM хранит данные логически как объекты разных типов. У объекта есть XML-представление.

Структура типов объектов и каталогов описывается в XMD-документах. XMD-файлы хранятся по шаблону:

```text
\wtv\wtv_<тип объекта>.xmd
```

Для каждого типа объекта существует как минимум один каталог, по которому возможен поиск объектов через `XQuery`. Структура каталога может отличаться от структуры полного XML-документа объекта.

Код в XMD-выражениях, например `OnBeforeSave`, нужно считать серверным кодом WebTutor во внутреннем серверном языке платформы.
