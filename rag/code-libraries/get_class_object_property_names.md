# GetClassObjectPropertyNames

`GetClassObjectPropertyNames( object )` возвращает список имен всех свойств объектов класса.

По источнику возвращается полный список имен, описанных в декларации класса, независимо от того, было ли конкретное свойство установлено в данном объекте.
