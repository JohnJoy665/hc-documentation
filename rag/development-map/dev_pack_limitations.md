# Ограничения dev-pack WebSoft HCM

Dev-pack содержит полезную карту разработки, но часть тем собрана частично:

- детальные функции `Tools`, `Tools_web`, `Tools_library`, `Ms_tools`;
- библиотеки программного кода;
- API v1 и API v3;
- XAML-элементы и атрибуты;
- редактор веб-страниц;
- функциональный дизайн;
- кэширование.

При разработке нельзя переносить конструкции Node.js/npm/browser API во внутренний серверный язык WebTutor без подтверждения локальными источниками.
