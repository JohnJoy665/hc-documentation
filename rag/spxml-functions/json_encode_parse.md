# EncodeJson и ParseJson

`EncodeJson(value, options)` преобразует базовый объект, массив или скалярное значение в JSON-строку.

Опции из источника:

- `PrettyFormat` - форматирование с отступами;
- `ExportLargeIntegersAsStrings` - экспортировать целые числа больше `0x7FFFFFFF` строками.
