# LoadUrlText, PutUrlText, UrlExists

Источник по URL-функциям перечисляет операции:

- `LoadUrlText()` и `LoadUrlData()` - чтение текста/данных по URL;
- `PutUrlText()` и `PutUrlData()` - запись текста/данных по URL;
- `UrlExists()` - проверка существования URL;
- `UrlFileSize()`, `UrlModDate()` - сведения о файле/ресурсе;
- `UrlToFilePath()`, `FilePathToUrl()` - преобразование между URL и путем.
